## 📋 Common Tasks Mapped to Tabs

### Task: New Employee Onboarding
1. **Assets** → Create laptop (LAP-010)
2. **Assets** → Assign to employee
3. **Dashboard** → Verify in asset count
4. **Reports** → Optional: Generate asset list

### Task: Equipment Replacement
1. **Assets** → Find old laptop (LAP-001)
2. **Assets** → Change status to Disposed
3. **Purchases** → Create PO for new laptop
4. **Assets** → Add new laptop (LAP-011)
5. **Reports** → Track replacement rate

### Task: Contract Renewal
1. **Dashboard** → See expiring contracts alert
2. **Contracts** → Click on expiring contract
3. **Contracts** → Click "Renew"
4. **Contracts** → Update renewal terms
5. **Dashboard** → Verify renewed status

### Task: Monthly Financial Review
1. **Dashboard** → Check key metrics
2. **Reports** → Generate spending report
3. **Reports** → Vendor-wise analysis
4. **Reports** → Export to Excel
5. **Presentations** → Present to leadership

### Task: Maintenance Planning
1. **Assets** → Filter assets in maintenance
2. **Contracts** → Check warranty coverage
3. **Reports** → Maintenance history
4. **Schedule** → Plan repairs
5. **Contracts** → Ensure coverage remains valid

## ⚡ Pro Tips

1. **Start with Dashboard** - Get overview before diving into details
2. **Use Filters** - Assets and Contracts have powerful filters
3. **Check Expiry Alerts** - Dashboard alerts prevent missed renewals
4. **Generate Reports Monthly** - Track trends over time
5. **Vendor Ratings** - Use to make better purchase decisions
6. **Export Reports** - Share with team for collaboration
7. **Auto-Generated Tags** - Don't worry about asset numbering
8. **Assignment Tracking** - Know who has what asset
9. **Status Tracking** - Mark items as maintenance to clear assignments
10. **Contract Renewals** - Renew before expiry to avoid gaps
