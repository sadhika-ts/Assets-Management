🐛 TROUBLESHOOTING:

Q: Backend won't start
A: Make sure port 5000 is free, or:
   $ lsof -i :5000
   $ kill -9 <PID>
   $ npm run dev

Q: Frontend shows blank page
A: Open DevTools (F12) and check console for errors
   Make sure backend is running
   Check .env file

Q: Need to reset database
A: $ npm run db:sync:fresh
   $ npm run db:seed
   $ npm run dev

Q: Getting import errors
A: cd client
   rm -rf node_modules
   npm install
   npm run dev
