📦 SEQUELIZE MODELS (6 Total)
═══════════════════════════════════════════════════════════════════════════════

1. ✅ User Model (models/User.js)
   ├─ Fields: id, name, email, password_hash, role, created_at
   ├─ Roles: admin, staff, viewer
   ├─ Associations: hasMany Asset, hasMany AuditLog
   └─ Table: users

2. ✅ Purchase Model (models/Purchase.js)
   ├─ Fields: id, purchase_id, vendor_name, vendor_contact, vendor_email,
   │           billing_address, shipping_address, purchase_date, total_amount, status
   ├─ Associations: hasMany Asset
   └─ Table: purchases

3. ✅ Asset Model (models/Asset.js)
   ├─ Fields: id, asset_tag, category, sub_type, serial_no, mac_address,
   │           status, purchase_id, assigned_to, created_at
   ├─ Categories: IT, Non-IT
   ├─ Status: active, inactive, disposed
   ├─ Associations: belongsTo Purchase, belongsTo User, hasOne AssetDetail, hasMany AuditLog
   └─ Table: assets

4. ✅ AssetDetail Model (models/AssetDetail.js)
   ├─ Fields: id, asset_id (unique), os_type, os_version, product_id, os_activated,
   │           processor_name, manufacturer, cores, ram_gb, disk_gb, disk_model,
   │           ms_office, office_key, software_list, configuration, others, created_at
   ├─ Associations: belongsTo Asset
   └─ Table: asset_details

5. ✅ Contract Model (models/Contract.js)
   ├─ Fields: id, contract_id, name, vendor_name, active_from, active_till, status, notes
   ├─ Status: active, expired, upcoming
   ├─ Associations: Standalone (no foreign keys)
   └─ Table: contracts

6. ✅ AuditLog Model (models/AuditLog.js)
   ├─ Fields: id, asset_id, user_id, action, old_value, new_value, changed_at
   ├─ Associations: belongsTo Asset, belongsTo User
   └─ Table: audit_logs

📊 DATABASE SCHEMA
═══════════════════════════════════════════════════════════════════════════════

USERS TABLE (users)
├─ id (UUID PK)
├─ name (String)
├─ email (String, UNIQUE)
├─ password_hash (String)
├─ role (ENUM: admin|staff|viewer)
└─ created_at (DateTime)

PURCHASES TABLE (purchases)
├─ id (UUID PK)
├─ purchase_id (String, UNIQUE)
├─ vendor_name (String)
├─ vendor_contact (String)
├─ vendor_email (String)
├─ billing_address (Text)
├─ shipping_address (Text)
├─ purchase_date (Date)
├─ total_amount (Decimal)
├─ status (String)
└─ created_at (DateTime)

ASSETS TABLE (assets)
├─ id (UUID PK)
├─ asset_tag (String, UNIQUE)
├─ category (ENUM: IT|Non-IT)
├─ sub_type (String)
├─ serial_no (String)
├─ mac_address (String)
├─ status (ENUM: active|inactive|disposed)
├─ purchase_id (UUID FK)
├─ assigned_to (UUID FK)
└─ created_at (DateTime)

ASSET_DETAILS TABLE (asset_details)
├─ id (UUID PK)
├─ asset_id (UUID FK, UNIQUE)
├─ os_type (String)
├─ os_version (String)
├─ product_id (String)
├─ os_activated (Boolean)
├─ processor_name (String)
├─ manufacturer (String)
├─ cores (Integer)
├─ ram_gb (Float)
├─ disk_gb (Float)
├─ disk_model (String)
├─ ms_office (Boolean)
├─ office_key (String)
├─ software_list (Text)
├─ configuration (Text)
├─ others (Text)
└─ created_at (DateTime)

CONTRACTS TABLE (contracts)
├─ id (UUID PK)
├─ contract_id (String, UNIQUE)
├─ name (String)
├─ vendor_name (String)
├─ active_from (Date)
├─ active_till (Date)
├─ status (ENUM: active|expired|upcoming)
├─ notes (Text)
└─ created_at (DateTime)

AUDIT_LOGS TABLE (audit_logs)
├─ id (UUID PK)
├─ asset_id (UUID FK, Nullable)
├─ user_id (UUID FK)
├─ action (String)
├─ old_value (Text)
├─ new_value (Text)
└─ changed_at (DateTime)
