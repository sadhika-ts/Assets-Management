## 🐛 Troubleshooting

### Backend won't start
```bash
# Check port 5000
lsof -i :5000

# Reset database
npm run db:sync:fresh
npm run db:seed
```

### Frontend shows blank
- Check DevTools console (F12)
- Make sure backend is running
- Check .env file

### Need clean install
```bash
cd client
rm -rf node_modules
npm install
npm run dev
```
